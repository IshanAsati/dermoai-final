Architecture

ConvNeXt-Base backbone with a custom 12-class classification head. Leverages a modernized ConvNet design incorporating depthwise convolutions, inverted bottlenecks, and large-kernel (7x7) depthwise convolutions inspired by Vision Transformers. The 12-class head uses a 2-layer MLP with hidden dimension 1024 and dropout of 0.3.

Training

Trained on HAM10000 + DermNet + Clinical datasets (7,000+ images) for 75 epochs using AdamW optimizer with cosine annealing learning rate schedule. Focal loss (gamma=2.0) addresses class imbalance across 12 conditions. Data augmentation includes random horizontal/vertical flips, rotations, color jitter, CutMix, and RandAugment.

Bias Calibration

Systematic bias towards melanoma under-prediction is counteracted via a logit bias calibration layer on the MEL output node. The bias value is tuned on a held-out validation set to optimize melanoma recall while preserving specificity. Current MEL recall: 71.1% (biased).

Inference Pipeline

3x test-time augmentation (TTA): center crop, horizontal flip, vertical flip. Probabilities are averaged across augmentations. Focal loss logits are converted via softmax. A calibrated MEL logit bias is applied pre-softmax when enabled. Full pipeline runs in under 2 seconds on a modern GPU.
